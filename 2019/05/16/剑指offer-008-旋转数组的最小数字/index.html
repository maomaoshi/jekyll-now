008-旋转数组的最小数字-C++

Created at 2019-05-16 Updated at 2019-05-16 Category 剑指Offer Tag 剑指Offer

题意

题目描述

把一个数组最开始的若干个元素搬到数组的末尾，我们称之为数组的旋转。 输入一个非减排序的数组的一个旋转，输出旋转数组的最小元素。 例如数组{3,4,5,1,2}为{1,2,3,4,5}的一个旋转，该数组的最小值为1。 NOTE：给出的所有元素都大于0，若数组大小为0，请返回0。牛客网链接

分析

1. 遍历数组寻找最小值

该思路很简单，就是遍历一遍数组，找出最小值。时间复杂度：O(n)

2. 最大值后面就是最小值

  • 根据观察可知，最大值后面就是最小值。如果序列旋转后，还是有序，则序列的第一个元素就是最小值。

3. 分治——二分查找

  • 用两个指针left, right分别指向数组的第一个元素和最后一个元素。把数组当作两个有序数组。
  • 如果左指针指向的元素小于右指针指向的元素，则该数组旋转后，仍是有序的。即左指针指向的元素是最小的。
  • 一般情况下，第一个元素是大于或等于最后一个元素。此时可得到数组中间的元素，且mid = (left + right)/2;
    • 如果中间元素位于前一个有序数组，即a[mid] >= a[left]，则把左指针指向该中间元素。
    • 如果中间元素位于后一个有序数组，即a[mid] <= a[right]，则把右指针指向该中间元素。
    • 所以，左指针总是指向第一个有序数组的元素，右指针总是指向第二个有序数组的元素。到最后，左右指针会指向两个相邻的元素，且右指针指向最小的那个数。
    • 如果a[left] == a[mid] && a[mid] == a[right]，则无法判断中间元素是属于前面一个还是后面一个元素，只能顺序查找。
