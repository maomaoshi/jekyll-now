LeetCode 136只出现一次的数字-C++

撰写于 2019-03-23 修改于 2019-03-23 分类 C++ / Leetcode 标签 C++ / LeetCode

题目

思路1：map结构

用map记录nums中元素出现的次数，然后遍历map，找到只出现一次的元素。

思路2：异或

虽然一开始注意到了题目要求不使用额外空间来实现，但是一时也想不到开辟另外空间来记录nums中元素出现的次数，还有什么其他方法。看了评论，居然是异或，顿时感觉自己是个菜鸡。异或有以下性质：

  1. 交换律：a ^ b ^ c <=> a ^ c ^ b;
  2. 任何数与0异或等于其本身： 0 ^ n => n;
  3. 相同的数异或结果为0：n ^ n => 0;
    所以int nums[3] = {2, 1, 2};
    2 ^ 1 ^ 2 = 2 ^ 2 ^ 1 = 0 ^ 1 = 1;
