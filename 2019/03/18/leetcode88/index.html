LeetCode第88题合并两个有序数组-C++

Created at 2019-03-18 Updated at 2019-03-18 Category C++ / Leetcode Tag C++ / LeetCode

题目

思路

一开始的想法是，定义两个变量p1,p2分别表示nums1和nums2的下标。如果nums1[p1] > nums2[p2]，则将nums2[p2]插入nums1[p1]。从头到尾遍历数组nums2，完成合并，类似于合并两个有序的链表。但是我忽略了一个问题，num1后面的元素为0。看了评论里的代码，可从尾到头合并数组。

class Solution {
public:
    void merge(vector<int>& nums1, int m, vector<int>& nums2, int n) {
        int p = m-- + n-- -1;
        while(m >= 0 && n >= 0){
            if(nums1[m] > nums2[n]){
                nums1[p--] = nums1[m--];
            }
            else{
                nums1[p--] = nums2[n--];
            }
        }
        while(n >= 0){
            nums1[p--] = nums2[n--];
        }

    }
};
