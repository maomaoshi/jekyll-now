LeetCode 142环形链表||-C++

撰写于 2019-03-26 修改于 2019-03-26 分类 C++ / Leetcode 标签 C++ / LeetCode

题目

思路1：set

遍历单链表，用set存储。如果单链表有环，就会在set中找到该结点。

思路2：快慢指针

起初fast,slow1都在同一起点，但是fast每次走两步，slow1走一步。如果链表有环，则两指针会在环内相遇。设链表起点到环入口点长度为a，快慢指针相遇的点距环入口点的长度为b。所以slow1走过的路程为(a+b)，fast走过的路程为2(a+b)。快慢指针相遇后，假设slow1要再走到相遇点(a+b)，还需要走多少路程呢？2(a+b) = a+b+c+b,c代表从相遇点走到环入口点的距离，所以c = a。即c等于链表起点到环入口点的长度。这是在用一个slow2指针从起点和slow1从相遇点(a+b)同时走，最终两个指针会在环入口点相遇。
