LeetCode 876链表的中间节点-C++

Created at 2019-03-26 Updated at 2019-03-27 Category C++ / Leetcode Tag C++ / LeetCode

题目

思路

初始化快慢指针fast,slow为head。每次fast走两步，slow走一步。当链表长度为奇数时，且当fast走到最后一个结点，即fast->next=nullptr时，slow位于中间结点。当链表长度为偶数时，且当fast=nullptr时，slow位于中间结点。可以举个例子，画图比较容易理解。
