LeetCode 160相交链表-C++

Created at 2019-03-26 Updated at 2019-03-27 Category C++ / Leetcode Tag C++ / LeetCode

题目

思路1：set

先将链表A的结点用set存储，再遍历链表B,若能在set中找到该链表B中的结点，则该结点为相交结点。

思路2：

定义两个指针pA，pB，如果pA到了链表A结尾，就指向链表B的头结点headB。如果pB到了链表B结尾，就指向链表A的头结点headA。如果两个链表有相交，两个指针会在相交点相遇。为什么呢？现在假设链表A头结点到相交点的距离为a，链表B头结点到相交点的距离为b。相交点之后链表长为c。所以pA走完链表A，走了a+c；pB走完链表B，走了b+c。然后pA,pB分别再指向headB,headA。pA从headB走到相交点走了b，pB从headA走到相交点走了a。所以两个指针走的路程a+c+b = b+c+a，最终再相交点相遇。如果链表没有交点，即c=0，两个指针都走了a+b。
