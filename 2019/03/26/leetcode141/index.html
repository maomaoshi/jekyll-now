LeetCode 141环形链表-C++

撰写于 2019-03-26 修改于 2019-03-26 分类 C++ / Leetcode 标签 C++ / LeetCode

题目

思路1：快慢指针

设置快慢指针，慢指针每次只走一步，快指针每次走两步。假设单链表有环，则快指针最后肯定会与慢指针相遇。假设单链表没有环，则快指针会到达链表的末尾，结束循环。

思路2：set

C++STL中set容器会自动对插入的元素进行排序，在Set中每一个元素都是唯一的。在这里用来存储链表中的每个结点，如果链表中存在环，则最终会在set中找到该元素。
