hexo+github搭建个人博客心得

Created at 2017-10-13 Updated at 2017-10-14 Category Blog Tag Blog

前言

10.2赶在去重庆之前,花了一天的时间把博客给搭好了。搭建的过程中也遇到了很多问题，写这篇博客就是想把搭建过程和遇到的问题和解决方法写下来，作为备忘和巩固吧。

安装hexo

  • 在安装hexo之前，必须安装：

    1.Node.js

    2.Git

    如果两个都安装好后就可以直接安装hexo

  • 在要安装hexo的路径下打开Git Bash输入
    $ npm install -g hexo-cli

  • 然后在要搭建博客的路径下，单击右键，打开Git Bash

    $ hexo init

    $ npm install

注册github账号

github官网

创建新仓库

  • 创建一个新的仓库，仓库名为 用户名.github.io
    比如：maomaoshi.github.io
    maomaoshi就是我github的用户名

配置SSH

  • 输入ssh-keygen -t rsa -C "注册的github邮箱"

  • 然后一路回车，之后会在本地~/.ssh生成id_rsa(密钥)和 id_rsa.pub（公钥）

  • 查看公钥里面的内容： cat ~/.ssh/id_rsa.pub

  • 将公钥的里的内容复制下来添加到github的SSH Key

  • 最后可以输入ssh -T git@github检测是否配置成功

修改config文件

  • 在目录下打开_config.yml修改config文件

最后

$ npm install hexo-deployer-git --save

$ hexo d -g

  • 这样就部署完成了，然后可以通过http://你的用户名.github.io访问自己博客
