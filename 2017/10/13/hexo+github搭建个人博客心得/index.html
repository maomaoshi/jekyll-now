hexo+github搭建个人博客心得

撰写于 2017-10-13 修改于 2017-10-13

前言

10.2赶在去重庆之前,花了一天的时间把博客给搭好了。搭建的过程中也遇到了很多问题，写这篇博客就是想把搭建过程和遇到的问题和解决方法写下来，作为备忘和巩固吧。

安装hexo

在安装hexo之前，必须安装：

1.Node.js

2.Git

如果两个都安装好后就可以直接安装hexo

在要安装hexo的路径下打开Git Bash输入

然后一路回车，之后会在本地~/.ssh生成id_rsa(密钥)和id_rsa.pub（公钥）

查看公钥里面的内容：

$ hexo d -g

这样就部署完成了，然后可以通过http://你的用户名.github.io访问自己博客
