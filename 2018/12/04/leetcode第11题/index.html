Created at 2018-12-04 Updated at 2018-12-06 Category C++ / Leetcode Tag C++ / LeetCode

题目

暴力法

双指针法

思路：我们在由线段长度构成的数组中使用两个指针，一个放在开始，一个置于末尾。 此外，我们会使用变量 max 来持续存储到目前为止所获得的最大面积。 在每一步中，我们会找出指针所指向的两条线段形成的区域，更新 max，并将指向较短线段的指针向较长线段那端移动一步。
