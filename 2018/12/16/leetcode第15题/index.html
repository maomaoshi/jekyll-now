LeetCode第15题三数之和-C++

Created at 2018-12-16 Updated at 2018-12-16 Category C++ / Leetcode Tag C++ / LeetCode

题目

思路

  1. 先将数组nums进行排序，然后从左往右依次枚举，在枚举的过程中左右夹逼。
  2. 枚举位置为i,、左、右位置分别用l、r表示，target = -nums[i],当nums[l]+nums[r]==target，则算找到一个，要存起来，再将左（右）边界往右夹逼同时要跳过左（右）边界相同的值，否则会出现重复的三个数；当nums[l]+nums[r]<target,左边界往右夹逼；否则，右边界往左夹逼。
  3. 在步骤2的整个过程中，始终要保证l<r。
