LeetCode第12题：整数转罗马数字-C++

撰写于 2018-12-05 修改于 2018-12-06 分类 C++ / Leetcode 标签 C++ / LeetCode

题目


思路

  1. 使用map来存储数值和字符之间的对应关系。
  2. 用除10取余法，从个位开始处理每位数，每位对应的真实值为余数乘以这一位上的权值pow(10,p)。
  3. 当余数r为4或9时，则特殊处理。
  4. 将每位对应的罗马字符串压入栈中
  5. 最后逐个弹出栈拼接成最终结果
