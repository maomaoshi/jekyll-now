C++之STL各容器的基本操作

Created at 2018-11-19 Updated at 2018-11-24 Category C++ Tag C++

在刷leetcode的时候，总是会用到STL的一些容器，比如vector, map, stack等。由于之前很少接触这些，导致在使用的时候很不熟练，所以在这里记录一下这些容器的基本用法。

0x00 vector

头文件：#include< vector >

定义：
vector< type > v1; //默认v1为空
vector< type > v2(n,i); //v2包括n个值为i的的type类型元素
vector< type > v3(n); //v3含有n个值为0的元素

基本操作：

  1. v.push_back(t) 在容器的最后加入一个值为t的数据
  2. v.size() 返回容器中元素的个数
  3. v.empty() 判断容器是否为空
  4. v.pop_back() 删除容器中的末元素，且不返回该值。
  5. v[n] 返回v中位置为n的元素
  6. v.clear() 删除容器中的全部元素

0x01 map

map的功能：自动建立key-value的对应。key和value可以是你任意需要的类型

头文件： #include< map >

定义：
map mapTemp;

基本操作：
mapTemp.count() //返回被查找元素的个数，如果有返回1，否则0
mapTemp[n] //访问key为n对应的value
mapTemp[m] = val //赋值

0x02 stack

头文件：#include < stack >

定义：
stack< type > s;

基本操作：
s.push(i) //将元素压入栈
s.top() //返回栈顶元素
s.pop() //将栈顶元素弹出栈，并不返回该值
s.empty() //返回该栈是否为空
s.size() //返回栈中元素的个数

0x03 set

关于set，必须说明的是set关联式容器。set作为一个容器也是用来存储同一数据类型的数据类型，并且能从一个数据集合中取出数据，在set中每个元素的值都唯一，而且系统能根据元素的值自动进行排序。应该注意的是set中数元素的值不能直接被改变。

头文件： #include < set >

定义：
set< type > s;

基本操作：
s.begin() //返回指向第一个元素的迭代器
*s.begin() //返回第一个元素的值
s.count() //返回某个值元素的个数
s.erase() //删除集合中的元素
s.insert() //在集合中插入元素
s.size() //集合中元素的数目
s.find() //返回一个指向被查找到元素的迭代器
s.end() //返回指向最后一个元素的迭代器
