LeetCode第七题：整数反转-C++

Created at 2018-11-30 Updated at 2018-11-30 Category C++ / Leetcode Tag C++ / LeetCode

题目

思路

flag记录x是负数还是整数，相应flag为-1或1。temp是x的绝对值。用除10取余法将temp每一位的值放到数组a[]中。每位乘以相对应的权值a[j]pow(10, i-j-1),结果和sum相加。如果溢出，则sum变成负数，返回0。最后sumflag就是最终结果。

官方题解


INT_MAX = 2147483647;
INT_MIN = -2147483648;
